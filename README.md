# Network Delay Analysis & Packet Tracer Simulation

A networking project focused on packet delay analysis, jitter calculation and network topology implementation using Cisco Packet Tracer.

## Features

* End-to-end delay calculation
* Jitter analysis using ping measurements
* Packet transmission simulation
* Router and switch configuration
* Network topology design
* Troubleshooting connectivity issues
* Performance analysis of simple LAN topologies

## Technologies

* Cisco Packet Tracer
* Networking Concepts
* Ping / Jitter Analysis
* Routing & Switching
* Delay Analysis

## Purpose

This project was developed as part of university coursework in Computer Networks to study real-world networking behavior, packet transmission delay and network performance optimization.
